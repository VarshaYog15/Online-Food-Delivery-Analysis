# 🍔 Online Food Delivery Data Analysis & Dashboard

## 📌 Project Overview

This project analyzes an online food delivery dataset to extract insights and build an interactive dashboard.
It covers **data cleaning, feature engineering, exploratory data analysis (EDA), and visualization** using Python.

---

## 🚀 Features

* Data Cleaning & Preprocessing
* Feature Engineering
* Exploratory Data Analysis (EDA)
* Interactive Dashboard using Streamlit
* MySQL Integration (optional)

---

## ⚙️ Installation

### 1. Clone the repository

```
git clone https://github.com/VarshaYog15/Online-Food-Delivery-Analysis.git
cd Online-Food-Deliver-Analysis
```

### 2. Create virtual environment (optional but recommended)

```
python -m venv venv
venv\Scripts\activate
```

### 3. Install dependencies

```
python -m pip install -r requirements.txt
```

---

## ▶️ How to Run

### Run EDA

```
python eda.py
```

### Run Dashboard

```
streamlit run food_dashboard.py
```
